# 페르소나 — 팀 일감 관리 앱 (MVP)

> MVP 헌장: 단일 테이블 · 기능 5개 이하 · 1일 안에 코드로 옮길 수 있는 분량

| 역할 | 기술 수준 | 주요 시나리오 |
|------|-----------|--------------|
| **팀장 (이서연, 34세)** 스프린트를 계획하고 일감을 팀원에게 배분하는 PM | 중급 — 스프레드시트·협업 도구에 익숙, 코드는 읽지 않음 | ① 매주 월요일 새 일감을 한 번에 여러 개 등록하고 담당자를 지정한다 ② 전체 보드를 훑어보며 병목(진행 중이 너무 많은 사람)을 파악한다 ③ 마감일이 지난 일감을 빠르게 식별해 담당자에게 알린다 |
| **개발자 (박지호, 28세)** 자신에게 할당된 일감을 처리하는 팀원 | 고급 — CLI·IDE 중심, 마우스 클릭 최소화 선호 | ① 오늘 내 일감만 필터링해서 우선순위 순으로 확인한다 ② 작업을 시작할 때 상태를 "할 일 → 진행 중"으로 즉시 변경한다 ③ 완료 후 상태를 "완료"로 바꾸고 다음 일감으로 넘어간다 |
| **파트타임 디자이너 (최수아, 26세)** 주 2–3회 접속하며 디자인 관련 일감만 담당 | 초급 — 피그마·노션 사용, 개발 도구 경험 없음 | ① 접속 시 내 이름으로 필터링해 지금 해야 할 일이 무엇인지 한눈에 파악한다 ② 작업 설명이 부족하면 일감 카드를 열어 상세 내용을 확인한다 ③ 작업이 끝나면 완료 처리 후 로그아웃 없이 탭을 닫는다 |
